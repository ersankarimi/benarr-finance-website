# Benarrr Finance Website

This project is a website for Benarrr Finance, built using Tailwind CSS for styling.

## Project Setup

To get started with the project, ensure you have [PNPM](https://pnpm.io/) installed as the package manager.

### Requirements

- Node.js (version >= 14.x)
- PNPM (version >= 9.12.1)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/benarrr-finance-website.git
   ```

2. Navigate into the project directory:

   ```bash
   cd benarrr-finance-website
   ```

3. Install the dependencies using PNPM:
   ```bash
   pnpm install
   ```

### Development

To start the development server and watch for changes in your CSS files, run:

```bash
pnpm dev
```

This will automatically rebuild the CSS file whenever changes are detected.

### Production Build

To create a production build, run:

```bash
pnpm build
```

This will generate the final CSS output in `./src/css/output.css`.

### Prettier

This project uses Prettier along with a Tailwind CSS plugin to format code. To format the files, you can run:

```bash
pnpm prettier --write .
```
